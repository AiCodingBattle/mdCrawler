# NextJS ​
NextJS is a React framework that enables functionality such as server-side rendering and generating static websites.
Example repository.
## Server build (NodeJS) ​
  * Set `Build Pack` to `nixpacks`.


## Static build (SPA) ​
  * Set `Build Pack` to `nixpacks`.
  * Enable `Is it a static site?`.
  * Set `Output Directory` to `out`.
